# 3Bros-WebCrawler
WebCrawler for 3Bros Cookies

WebCrawler to gather store information from Food Lions stores across the US to add to 3Bros Cookies' company website.

Gathered...
- Location name
- Street Address
- City
- State
- Zip code
- Phone number
- Latitude
- Longitude
- Country
- Website URL

Saved 1,141 locations
